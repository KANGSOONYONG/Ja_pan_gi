# Ja_pan_gi

Android Studio를 이용해서 자판기 프로그램을 만들어보았습니다.

투입된 금액, 제품 금액은 사용자에게 입력받고, "계산하기" 버튼을 누르면 잔액과 거스름돈을 알려줍니다.
